# Budget App Masonry UI

### Built With

* Bootstrap 4.0
* Sass 3 [Syntax:SCSS]
* Masonry.js
* Google Material Design

### Installation

1. Clone the repo
```sh
git clone https://github.com/your_username_/Project-Name.git
```
2. Install NPM packages
```sh
npm install
```
3. SCSS Auto Compile Script
```sh
npm run compile:sass
```
